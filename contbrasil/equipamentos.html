Nossos Containers

Refrigerado
Carga Seca

Uso

O container refrigerado é utilizado principalmente para a armazenagem de produtos que precisam de controle de temperatura, como congelados e resfriados. Também é possível trabalhar com temperaturas positivas.

Tamanhos (MEDIDAS EXTERNAS)

12m comprimento x 2,43m largura x 2,89 altura;

12m comprimento x 2,43m largura x 2,59 altura;

6m comprimento x 2,43m largura x 2,59 altura;

3m comprimento x 2,43m largura x 2,59 altura;

Voltagens

Os containers têm maquinário próprio e funcionam com voltagem de 220v, 380v ou 440v.

Temperatura

Pode ser operado com temperaturas de aproximadamente 25°C a -25°C. (se necessário, com pequenas modificações, pode se obter temperaturas maiores ou menores).

Customização

Os containers refrigerados são disponibilizados conforme o espaço e necessidade do cliente, podendo ser adaptados com iluminação, alarmes, cortinas além de adaptações de portas na lateral.

Uso

Para serem utilizados como armazém em canteiros de obras, indústrias e outros segmentos de negócios com necessidade de espaço temporário para guarda de produtos diversos, nós indicamos o Container de Carga Seca, que também pode ser utilizado para armazenagem de cargas que não apresentam necessidade de controle de temperatura.

Tamanhos (MEDIDAS EXTERNAS)

12m comprimento x 2,43m largura x 2,89 altura;

12m comprimento x 2,43m largura x 2,59 altura;

6m comprimento x 2,43m largura x 2,59 altura;

3m comprimento x 2,43m largura x 2,59 altura;

Customização

Os equipamentos para armazenar produtos secos podem ser adaptados com janelas e portas de acordo com a necessidade do cliente.
